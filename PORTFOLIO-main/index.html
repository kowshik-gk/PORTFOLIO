Project:DONATIFY

<

Domain of this project is Web development,the created website is usefull to donate the food which excess from family functions ,parties to the Organizations of needy people.Here we are not going to deliver a food but we make connection between donor and receiver of the food

The Name of Site is DONATIFY ,The website for food donation acts as an interface between the users who are looking for a channel to give the excess food without wasting it. It enables us to donate the excess food by notifying nearby users with the details of the food that is available. The required users claim the notification. The system allocates the food items based on the priority

For frontend:HTML,CSS only used
For Database:MY SQLis used
For Validation : Javascript were used

If the user is looking to donate food, then the user must login using username and password then add the below information in the application:

  • Name of the food item and the excess quantity.
  • Contact address details to claim excess food
  • The Main Objective of this website is to make connection between FOOD DONOR and FOOD RECEIVER
